## Project Overview
This project is an Excel dashboard built to analyze cryptocurrency market data.

- Average Price
- Total Volume
- Market Capitalization
- High & Low Price

## Charts Used
- Line Chart (Price Trend)
- Area Chart (Market Cap Trend)
- Column Chart (Volume)
- KPI Cards

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers & Timeline
